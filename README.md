# Digital Art Marketplace

A premium P2P platform for artists and collectors to trade digital art using UPI and Indian Rupees (₹).

## Features
- **UPI Gateway:** High-fidelity transaction verification with UTR and Transaction ID matching.
- **P2P Manual Verification:** Sellers can manually confirm payments for absolute security.
- **Real-Time Presence:** See when artists are online.
- **Messenger:** Built-in chat system for direct negotiation.
- **Resale Market:** Buy art and relist it for a profit.

## Tech Stack
- React
- Firebase (Auth, Firestore, Storage)
- Vite
- Glassmorphism UI (Vanilla CSS)
